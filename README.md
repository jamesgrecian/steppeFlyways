# steppeFlyways
 
*Flyway-scale analysis of Steppe Eagle migration and threat exposure.*
 
## Overview
 
This repository hosts the code and analysis pipeline for a flyway-scale study of Steppe Eagle (*Aquila nipalensis*) migration, supporting implementation of the CMS Raptors MOU Single Species Global Action Plan (GAP) adopted at COP15 in March 2026.
 
The analysis combines satellite telemetry (via Movebank) with threat layers covering power infrastructure, wind energy, and land use to map migration corridors, stopovers, and wintering areas, and to identify potential risk hotspots along the Central Asian and East African flyways.
 
This is a working repository. Outputs will be staged and versioned as additional telemetry datasets are incorporated under MoUs with data holders.

## Getting started
 
This repository contains analysis code only. Telemetry data is held separately and access is governed by Memoranda of Understanding (MoUs) with individual data holders.
 
To run the pipeline you will need:
 
- R (≥ 4.4)
- Movebank credentials with relevant project access
- A signed MoU for any dataset you wish to use
- Local copies of the threat layer inputs (see `data/README.md`)
